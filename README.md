# Cloud Resume Frontend

This repository contains the frontend for my Azure Cloud Resume Challenge project.

## Overview

This site is a personal cloud portfolio built with:
- HTML
- CSS
- JavaScript

It is designed to be deployed as an Azure Storage Static Website and later integrated with:
- Azure CDN
- Azure Functions
- Cosmos DB
- GitHub Actions CI/CD

## Purpose

The goal of this project is to demonstrate hands-on cloud engineering skills through a production-style resume website.

## Current Status

Frontend design and local build complete. Azure deployment in progress.
